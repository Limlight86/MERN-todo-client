# Mern Todo Client

## Setup

### API
- `git clone git@github.com:wyncode/todo-mern-api.git`
- `cd todo-mern-api`
- `yarn`
- `cp sample.env .env`
- Fill in the environment variables.
- `yarn dev`

### Client
- `git clone https://github.com/Limlight86/MERN-todo-client`
- `cd todo-mern-client`
- `yarn`
- `cp sample.env .env`
- Fill in the environment variables.
- `yarn start`

### Dependencies
    "axios"
    "bootstrap"
    "moment"
    "moment-range"
    "react"
    "react-dom"
    "react-router-dom"
